Quick guide

  • Highlight / Pen ГўвӮ¬вҖқ drag on the page. Each has preset colors plus a rainbow chip for any custom color. With the Hand tool, click a highlight or stroke to select it: drag to move, ГўЕ“вҖў or Delete to erase, corners resize a highlight.
  • Text ГўвӮ¬вҖқ click, type, then Enter. Hebrew and other languages are supported (right-to-left is handled automatically). To change the words later: with the Text tool click the text and retype, or with the Hand tool double-click it. Select it with the Hand tool to change its colour or size from the bar at the top; Hand tool also moves it and resizes by the corners.
  • Image ГўвӮ¬вҖқ drag an image file straight onto the page, or use the Image tool and drag a box. With the Hand tool, drag to move or use the corners to resize.
  • Sign ГўвӮ¬вҖқ draw a signature once, then click on the page to place it.
  • Move around ГўвӮ¬вҖқ drag the page with the Hand tool, or hold the right mouse button with any tool. Ctrl + wheel zooms.
  • Pages ГўвӮ¬вҖқ drag thumbnails to reorder, or use a thumbnail's ГўвҖЎвҖҰ to jump it to any page number. Each has rotate & delete. Ctrl+click (or Shift+click) thumbnails to select several pages, then move them together (drag or "MoveГўвӮ¬ВҰ") or delete them in one go.
  • Insert PDF adds other PDFs at the beginning, end, or after a chosen page ГўвӮ¬вҖқ you can pick several files at once in the dialog, or simply drop PDF files onto the window to append them.
  • Extract saves a page range to a new file.

Private by design

Your PDFs are opened, edited, and saved entirely in this browser ГўвӮ¬вҖқ nothing is ever uploaded. This tool adds text as overlays; it does not rewrite text already inside the PDF. "Sign" places a visible signature image, not a cryptographic digital signature.
